## 需求概述
我想基于 tarui + vue3 + typescript 开发一个桌面应用程序，实现一个递归读取本地文件夹中的 excel 文件，并且按照指定的列名映射配置将 excel
列名进行转换，保存到另外一个工作目录中，该工作目录还维持原来的目录结果。转换完成后，调用指定的 http 接口逐个将 excel 文件上传到服务器上。
一些具体要求如下：

1. 列名映射在程序页面上添加，支持添加多组，每一组左边填写原列名，右边填写新列名，也支持删除。
2. 上传 excel 的接口会校验 token，页面上还需要填写服务器的域名、账号、密码，在开始上传的时候先调用登录接口获取到 token，然后再调用上传接口。
3. excel 列名的转换和上传分为两个操作进行，转换和上传的过程中在页面上显示相应的日志，例如：开始转换 excel，检测多多少个 excel，现在开始转换 xxxx.xlsx 等。
日志还需要保存到文件中。
4. 页面 UI 风格要符合现在桌面应用设计，选择适合的组件库，可以考虑 tailwindcss + NaiveUI 是否可以实现。
5. excel 的处理过程需要注意效率，采用流式处理，避免一下子将整个 excel 加载到内存，excel 文件可能会很大，达到大几百MB。
